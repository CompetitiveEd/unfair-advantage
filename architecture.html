Known Limitations

Source Quality

If you feed it marketing collateral instead of verified 10-Ks and raw transcripts, the outputs will be generic and useless.

Countermeasure: Strict input validation at every prompt node.

Human Layer

The engine can map the influence structure perfectly, but a human still has to walk into the room, hold eye contact, and execute the strategy.

Countermeasure: Behavioural science coaching built into the methodology.

Compute Discipline

Unconstrained LLM usage on junior tiers burns API budget rapidly. The architecture requires targeted, conditional prompts — not open-ended queries.

Countermeasure: Tiered access with prompt governance per role.

Intelligence Decay

Outputs lose their edge if source feeds aren't refreshed quarterly. Stale transcripts mean stale politics and hidden boardroom blind spots.

Countermeasure: Automated refresh triggers on earnings cycle.
