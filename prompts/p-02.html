Architecture

Production engine. Assembles a structured account plan from GAIN outputs. Three modes: FULL (12-month strategic plan), RENEWAL (renewal-focused), EXPANSION (growth-focused). Audience-ready document the operator can present to their leadership.

Engine Logic

IF G-series + N-series → FULL strategic plan. IF N-06 renewal data → RENEWAL plan. IF N-01 expansion signals → EXPANSION plan. Graceful degradation from available intelligence.

Power of "NONE"

G-01 = None → no macro context, flag critical. G-03 = None → no stakeholder depth, role-based only. N-series = None → current-state plan only, no growth strategy. Plan Horizon = None → default 12 months.
