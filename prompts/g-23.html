Architecture

Designs the attack vector to get a meeting with a specific person. Selects the medium (physical mail, email, LinkedIn, warm intro, event), the sensory format (cotton envelope, Japanese washi paper, handwritten note, printed QR code to bespoke microsite), the psychological framing (power position, subservient, peer-to-peer, provocation), and the content calibrated to the target's profile — what they'd respond to, what they'd ignore, what would make them curious enough to reply.

Engine Logic

IF G-03 profile available → calibrates medium, tone, and framing to target's modality, Kegan stage, and GAIN driver. IF G-22 waterfall available → uses consolidated intelligence to identify the hook — the one thing that would make this person stop and pay attention. Generates: recommended approach channel, physical format specification, message draft (multiple tone variants), QR/microsite content brief, and follow-up sequence.

Power of "NONE"

G-22 = None → uses G-03 profile only, narrower hook options. G-03 = None → generic role-based approach using industry patterns, no psychological calibration. Both = None → refuses to run. Blind approach without intelligence is spam.

Routing
Dual path — physical approach design + digital microsite brief.
