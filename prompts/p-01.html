Architecture

Production engine. Takes any GAIN outputs and compresses into audience-calibrated dashboards. Three modes: AUDIT (score existing dashboard), BUILD (create from GAIN outputs), COMPRESS (reduce any output to one-page). Tufte information design. SCQA structure. MECE classification. Audience-aware formatting.

Engine Logic

IF existing dashboard → AUDIT mode (score and fix). IF GAIN outputs → BUILD mode (create dashboard). IF long output → COMPRESS mode (one-page). Audience determines depth: Board (headline), Director (summary), Manager (detail), Analyst (full data).

Power of "NONE"

Source Outputs = None → cannot build, prescribe what to run. Audience = None → default to Director level. Existing Dashboard = None → BUILD mode. Format = None → default Markdown tables.
