Architecture

Two-layer engine. Layer 1: static relationship map — alliances, rivalries, dependencies, mentoring, gatekeeping, coalition opportunities, decision cascade paths. Layer 2: live operational intelligence — cross-meeting contamination risk (what you said to Person A that could backfire with Person B), borrowed authority audit (whose name you can safely reference with whom), narrative consistency check (are you telling the same story to everyone or have versions drifted), and meeting sequence intelligence (given you're seeing these people in this order, here's what to carry forward and what to bury).

Engine Logic

IF multiple G-03s with relational cross-refs → deep dynamic mapping. Identifies stable alliances, fragile alliances, and exploit points. Models decision cascade paths. IF Meeting Log provided → activates Layer 2: scans all prior conversations for contamination risk, flags language borrowed from one stakeholder that would land badly with another, checks narrative consistency across all touchpoints. IF Meeting Sequence provided → generates per-meeting contamination brief: before you walk in, here's what you said to others that matters.

Power of "NONE"

A-05 = None → deduces from G-03 relational data only. Meeting Observations = None → relies on profile analysis only. Meeting Log = None → Layer 2 disabled, static map only. Meeting Sequence = None → no forward-looking contamination brief.

Routing
Dual path — static dynamics map + live contamination/consistency layer.
