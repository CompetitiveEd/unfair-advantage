PRACTICAL GUIDANCE

Choosing Your Tools

The GAIN prompts work in any AI system that takes natural language. Different tools surface different things ‚Äî that's a feature, not a bug. Start free, upgrade when outputs go generic, and remember: the prompt is the methodology, not the platform.

DEPTH vs. SPEED

Depth for research, profiling, and briefings (Steps 1, 3, 6). Speed for lookups and fact-checking. You'll learn which tools suit which within days.

DIFFERENT TOOLS, DIFFERENT FINDS

Same prompt, two systems, different outputs. One surfaces LinkedIn activity, the other finds a podcast interview. Use multiple tools when stakes are high.

FREE vs. PAID

Start free. Every prompt works. Upgrade when outputs go generic or the tool starts forgetting context ‚Äî not before.

PLATFORM-INDEPENDENT

If your tool disappears tomorrow, the prompts still work. The methodology is the thinking framework, not the platform.

BEFORE YOU START

Read the output critically, not gratefully. AI produces confident, well-structured text regardless of accuracy. Your job is to verify, not to be impressed.

Save your outputs. Copy them into a document or note. AI conversations can be lost, context windows fill up, platforms reset. Your research is valuable. Treat it that way.

When The AI Conversation Breaks Down

Every AI conversation has a shelf life. When the context window fills, the system silently drops earlier content ‚Äî it doesn't warn you. Watch for repetition, forgotten context, generic output, or new 'facts' you never provided, then hand off to a fresh chat using the transfer prompt below.
