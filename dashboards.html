System Command: Generate.Visualisation();

These exact AI prompts translate unstructured data into the interactive HTML/CSS arrays shown above. Copy and run them directly in your LLM window alongside your GAIN research.

VP-01
Act as a Data Visualisation Architect. Take the data from my 'Authority Map' output. Generate an HTML/CSS scatter plot where: X-Axis is 'Influence' (Low-High), Y-Axis is 'Authority' (Low-High). Plot each stakeholder as a dot. Color code dots: Blue (Unknown), Amber (Targeted), Green (Established).
VP-02
Create a CSS Flexbox Org Chart based on the Stakeholder Register. For each node box, include Name, Role, and a color-coded status bar indicating Relationship Strength (Red/Amber/Green).
VP-03
Generate a CSS Grid Gantt Chart showing the 12-week engagement plan. Rows = Stakeholders. Columns = Weeks. Plot the specific engagement actions (Intro, Workshop, Proposal) as colored bars.
VP-04
Create a Stacked Bar Chart in CSS. Each bar represents an Evaluation Criterion (e.g., Technical, Commercial). The segments of the bar should represent which stakeholder influences that score, proportional to their authority.
VP-05
Create a Bubble Chart where Bubble Size = Evaluation Weighting (%). Color = Our Confidence Level (Green=High, Red=Low). Label each bubble with the Criterion Name.
VP-06
Generate a horizontal timeline map. Mark key dates (Submission, Orals, Sign) as vertical lines. Show 'Compression Zones' where multiple deadlines overlap or duration is short.
VP-07
Create a Bar Chart breaking down the deal scope by Service Tower. Height = Cost/Value. Color = TUPE status (Blue=Transfer, Green=New Scope).
VP-08
Create a 'Money Flow' diagram using CSS cards connected by arrows. Show Base Contract value, Risk Pot, Gain Share mechanism, and Innovation Fund.
VP-09
Create a Heatmap Table. Rows = Win Themes. Columns = Touchpoints (Exec Summary, Tech Response, Orals). Cells = Deployment Strength (Strong/Weak/Absent).
VP-10
Create a Network Diagram (Node Link). Central node = 'The Deal'. Surrounding nodes = Key Stakeholders. Connectors = Influence Lines (Green=Support, Red=Blocker). Identify the 'Linchpin' node.
VP-11
Create a Comparative Heatmap. Columns = Us vs Competitor A vs Competitor B. Rows = Evaluation Criteria. Color code cells by win probability.
VP-12
Create a White Space Analysis Grid. Rows = Service Lines. Columns = Business Units. Mark cells as 'Incumbent' (Red), 'Us' (Green), or 'Open Opportunity' (White).
VP-13
Generate a 90-Day Campaign Gantt Chart. Group rows by stream: Relationship, Service, Opportunity, Defence. Mark key milestones as diamonds.
VP-14
Create a One-Page CSS Dashboard Grid. Panels: Account Snapshot, Financial Performance, Top 3 Risks, Top 3 Opportunities, 90-Day Actions. Use RAG (Red/Amber/Green) indicators.
